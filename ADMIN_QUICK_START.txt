## ğŸ› Troubleshooting

### "Incorrect username or password"
- Check credentials match: `admin` / `admin123`
- Try backup account: `root` / `root123`

### "Unauthorized" error
- You're not logged in or session expired
- Go to / and login again

### Dashboard not loading data
- Check if login_dataset.csv exists
- Ensure http_requests.csv has write permissions

### Flask not starting
- Check port 5000 is not in use
- Run: `netstat -ano | findstr :5000`
- Kill process if needed and restart
